# Output Files

This directory contains all generated outputs from the analysis pipeline.

## Structure

### charts/
Visualization outputs in PNG and SVG formats.
- `top_roles_bar.png` - Top 20 in-demand job roles
- `skills_frequency.png` - Top 30 required skills
- `skills_heatmap.png` - Skills by job category matrix
- `geographic_map.png` - Jobs by location
- `work_type_pie.png` - Work type distribution
- `wordcloud_descriptions.png` - Job description word cloud

### reports/
Final reports and presentations.
- `job_trends_report.pdf` - Comprehensive PDF report
- `analytics_workbook.xlsx` - Excel workbook with multiple sheets
- `presentation.pptx` - Project presentation slides

### api/
JSON data for programmatic access.
- `summary.json` - Key metrics and statistics
- `top_skills.json` - Top skills with frequencies
- `top_companies.json` - Top hiring companies
- `location_stats.json` - Geographic statistics

### temp/
Temporary files (excluded from version control).
- Intermediate processing files
- Cache files
- Debug outputs

## File Naming Convention

- Charts: `{chart_type}_{description}.{format}`
- Reports: `{report_type}_{date}.{format}`
- API: `{endpoint_name}.json`

## Access

All output files are automatically generated by running the pipeline scripts.
Use the VS Code task **Run Full Pipeline** to regenerate all outputs.
